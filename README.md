A basic machine learning web app to predict Mumbai rainfall using historical data. Built with Streamlit and Decision Tree Regression, it offers an interactive interface to visualize and understand rainfall trends.

## Features

- Predicts rainfall based on historical patterns
- Uses Decision Tree Regression for modeling
- Clean and interactive Streamlit UI
- Simple data visualization and insights

## Tech Stack

- Python
- Streamlit
- scikit-learn
- pandas, matplotlib

## How It Works

1. Loads and visualizes past Mumbai rainfall data
2. Trains a basic Decision Tree Regressor on the dataset
3. Allows users to adjust parameters and predict future rainfall
4. Displays trend charts and predictions live on the interface
